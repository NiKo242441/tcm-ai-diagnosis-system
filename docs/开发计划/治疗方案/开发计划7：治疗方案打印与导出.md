# 开发计划7：治疗方案打印与导出

## 1. 功能定位与业务边界

治疗方案打印与导出功能是治疗流程的最终交付环节，**仅处理已完成版本管理并定稿的治疗方案的输出工作**。该功能不涉及时治疗方案内容的生成、编辑、禁忌检查等前置业务流程，仅聚焦于将已最终确认的治疗方案以标准化格式输出。

### 1.1 前置业务依赖（系统级标准 v4.0）
- **状态要求**：必须为TreatmentStatus.Finalized(6)（与计划1统一状态枚举100%匹配）
- **版本要求**：必须为V{主版本号}.{次版本号}.{修订号}格式（与计划1和计划5完全统一）
- **检查要求**：必须通过计划4配伍禁忌检查且状态为Versioning(5)流转至Finalized(6)
- **审计要求**：必须包含计划6完整审计日志（计划2→3→4→5→6→7生命周期轨迹）
- **数据要求**：必须整合计划1-6所有模块的六大治疗体系完整数据结构
- **权限要求**：必须基于CreatedByUserId统一字段验证用户权限（与计划1实体定义一致）

#### 1.3.2 数据一致性验证（系统级标准 v4.0）
```csharp
// 打印前生命周期完整性验证
public class PrintValidationService
{
    public async Task<ValidationResult> ValidatePrintDataAsync(long treatmentId)
    {
        var treatment = await _treatmentRepository.GetByIdAsync(treatmentId);
        var validation = new ValidationResult();
        
        // 1. 状态验证（必须与计划1统一枚举值6）
        if (treatment.Status != TreatmentStatus.Finalized)  // 枚举值6
        {
            validation.AddError($"状态必须为Finalized(6)，当前状态：{treatment.Status}({(int)treatment.Status})");
            return validation;
        }
        
        // 2. 版本格式验证（计划5统一规范：V{主}.{次}.{修订}）
        if (!VersionNumberGenerator.IsValidFormat(treatment.Version))
        {
            validation.AddError("版本号格式必须为V{主版本号}.{次版本号}.{修订号}");
            return validation;
        }
        
        // 3. 生命周期完整性验证（计划2→3→4→5→6→7）
        var auditLogs = await _auditRepository.GetTreatmentLifecycleAsync(treatmentId);
        var requiredModules = new[] { "计划2", "计划3", "计划4", "计划5", "计划6" };
        
        foreach (var module in requiredModules)
        {
            if (!auditLogs.Any(log => log.TriggerModule == module))
            {
                validation.AddError($"缺少{module}的审计记录");
            }
        }
        
        // 4. 数据完整性验证（六大治疗体系）
        var treatmentData = await _treatmentRepository.GetCompleteTreatmentAsync(treatmentId);
        var systems = new[] { "Herbs", "Acupuncture", "Moxibustion", "Cupping", "Lifestyle", "Followup" };
        
        foreach (var system in systems)
        {
            if (!treatmentData.HasSystemData(system))
            {
                validation.AddError($"缺少{system}治疗体系数据");
            }
        }
        
        // 5. 权限验证（CreatedByUserId统一字段）
        var currentUserId = _currentUserService.GetCurrentUserId();
        if (treatment.CreatedByUserId != currentUserId && !await _permissionService.HasExportPermissionAsync(currentUserId))
        {
            validation.AddError("无导出权限");
        }
        
        return validation;
    }
}
```

### 1.2 功能边界声明（系统级标准 v4.0）

**本模块严格遵循**以下边界原则：
- ✅ **仅处理Finalized状态**：仅处理计划5输出的`TreatmentStatus.Finalized(6)`状态的治疗方案
- ✅ **前置依赖验证**：必须完成计划1-6的全部前置处理流程（状态流转：1→2→3→4→5→6）
- ✅ **版本一致性**：仅处理最新版本（IsLatest = true）的治疗方案，版本号格式V{主}.{次}.{修订}
- ✅ **格式标准化**：提供PDF、Word、Excel三种标准导出格式（计划7强制标准）
- ✅ **中医特色模板**：内置标准化中医治疗方案模板（与计划1数据结构完全匹配）
- ✅ **数据完整性**：确保六大治疗体系完整导出（与计划1实体类完全匹配）
- ✅ **状态枚举验证**：严格按照计划1定义的枚举值1-7进行状态验证
- ❌ **不处理状态变更**：不修改、不生成、不编辑治疗方案状态（仅读取状态6）
- ❌ **不处理版本管理**：版本号由计划5统一处理，仅使用最终版本（IsLatest=true）
- ❌ **不处理内容编辑**：治疗方案内容仅读取，不修改（只读模式）
- ❌ **不处理权限控制**：用户权限由计划3统一验证（基于CreatedByUserId字段）

#### 3.2.1 经典方剂引用格式
```csharp
public class ClassicFormulaExportFormatter
{
    public string FormatClassicFormulaReferences(TreatmentData treatment)
    {
        var references = new StringBuilder();
        
        if (!string.IsNullOrEmpty(plan.ConfirmedSyndrome.TreatmentPrinciple))
        {
            references.AppendLine("## 经典方剂参考");
            references.AppendLine();
            
            // 从治疗原则中提取经典方剂引用
            var formulas = ExtractClassicFormulasFromTreatmentPrinciple(plan.ConfirmedSyndrome.TreatmentPrinciple);
            foreach (var formula in formulas)
            {
                var formulaInfo = GetFormulaDetails(formula.Trim());
                references.AppendLine($"### {formulaInfo.Name}");
                references.AppendLine($"**来源：** {formulaInfo.Source}");
                references.AppendLine($"**组成：** {formulaInfo.Composition}");
                references.AppendLine($"**功效：** {formulaInfo.Effects}");
                references.AppendLine($"**适用证候：** {formulaInfo.Indications}");
                references.AppendLine($"**化裁说明：** {formulaInfo.Modifications}");
                references.AppendLine();
            }
        }
        
        return references.ToString();
    }
    
    private List<string> ExtractClassicFormulasFromTreatmentPrinciple(string treatmentPrinciple)
    {
        // 从治疗原则中提取经典方剂名称
        var classicFormulas = new List<string> { "桂枝汤", "麻黄汤", "柴胡疏肝散", "四君子汤" };
```

```csharp
public class TcmExcelExporter

    public byte[] ExportToExcel(TreatmentData treatment)

        basicSheet.Cells["A3"].Value = "主诊医师"; basicSheet.Cells["B3"].Value = plan.DoctorInfo.Name;
        basicSheet.Cells["A4"].Value = "中医证型"; basicSheet.Cells["B4"].Value = plan.ConfirmedSyndrome.SyndromeName;
        basicSheet.Cells["A5"].Value = "治疗原则"; basicSheet.Cells["B5"].Value = plan.ConfirmedSyndrome.TreatmentPrinciple;
        
        // 中药处方工作表
        var herbalSheet = package.Workbook.Worksheets.Add("中药处方");
        herbalSheet.Cells["A1"].Value = "处方名称";
        herbalSheet.Cells["B1"].Value = "药材名称";
        herbalSheet.Cells["C1"].Value = "剂量";
        herbalSheet.Cells["D1"].Value = "单位";
        herbalSheet.Cells["E1"].Value = "炮制方法";
        herbalSheet.Cells["F1"].Value = "功效";
        
        int row = 2;
        foreach (var herb in plan.HerbalPrescription.Medicines)
        {
            herbalSheet.Cells[$"A{row}"].Value = plan.HerbalPrescription.PrescriptionName;
            herbalSheet.Cells[$"B{row}"].Value = herb.Name;
            herbalSheet.Cells[$"C{row}"].Value = herb.Dosage;
            herbalSheet.Cells[$"D{row}"].Value = herb.Unit;
            herbalSheet.Cells[$"E{row}"].Value = herb.Preparation;
            herbalSheet.Cells[$"F{row}"].Value = herb.Effects;
            row++;
        }
        
        // 针灸穴位工作表
        var acupSheet = package.Workbook.Worksheets.Add("针灸穴位");
        acupSheet.Cells["A1"].Value = "穴位名称";
        acupSheet.Cells["B1"].Value = "国际标准代码";
        acupSheet.Cells["C1"].Value = "针刺方法";
        acupSheet.Cells["D1"].Value = "留针时间(分钟)";
        acupSheet.Cells["E1"].Value = "频次";
        acupSheet.Cells["F1"].Value = "备注";
        
        row = 2;
        foreach (var point in plan.AcupuncturePoints.Points)
        {
            acupSheet.Cells[$"A{row}"].Value = point.Name;
            acupSheet.Cells[$"B{row}"].Value = GetStandardAcupointCode(point.Name);
            acupSheet.Cells[$"C{row}"].Value = point.Technique;
            acupSheet.Cells[$"D{row}"].Value = point.RetentionTime;
            acupSheet.Cells[$"E{row}"].Value = point.Frequency;
            acupSheet.Cells[$"F{row}"].Value = point.Notes;
            row++;
        }
        
        return package.GetAsByteArray();
    }

            ["内关"] = "PC6",
            ["三阴交"] = "SP6",
            ["太冲"] = "LR3",
            ["百会"] = "GV20",
            ["关元"] = "CV4"
        };
        
        return codeMap.ContainsKey(pointName) ? codeMap[pointName] : pointName;
    }
```
