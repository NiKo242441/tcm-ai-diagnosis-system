## 📝 权限管理最佳实践

### 1. 最小权限原则
- 只授予用户完成工作所需的最小权限集
- 定期审查和清理不必要的权限

### 2. 角色分离
- 明确区分不同角色的职责边界
- 避免权限过度集中

### 3. 临时权限
- 对于特殊场景（会诊、转诊），使用临时权限
- 设置明确的有效期

### 4. 审计追踪
- 所有权限验证失败都会记录到审计日志
- 定期检查异常访问尝试
