## 🔧 常用命令

### Entity Framework 迁移
```powershell
# 创建迁移
dotnet ef migrations add <MigrationName> --project TcmAiDiagnosis.EFContext --startup-project TcmAiDiagnosis.Web

# 应用迁移
dotnet ef database update --project TcmAiDiagnosis.EFContext --startup-project TcmAiDiagnosis.Web

# 回滚迁移
dotnet ef database update <PreviousMigrationName> --project TcmAiDiagnosis.EFContext --startup-project TcmAiDiagnosis.Web

# 删除最后一个迁移
dotnet ef migrations remove --project TcmAiDiagnosis.EFContext --startup-project TcmAiDiagnosis.Web

# 查看迁移列表
dotnet ef migrations list --project TcmAiDiagnosis.EFContext --startup-project TcmAiDiagnosis.Web
```

### 数据库操作
```sql
-- 查看权限数量
SELECT COUNT(*) FROM permissions;

-- 查看各分类权限数量
SELECT category, COUNT(*) as count 
FROM permissions 
GROUP BY category 
ORDER BY count DESC;

-- 查看角色权限分配
SELECT r.Name as RoleName, COUNT(rp.permission_id) as PermissionCount
FROM AspNetRoles r
LEFT JOIN role_permissions rp ON r.Id = rp.role_id
GROUP BY r.Id, r.Name;

-- 查看某个角色的所有权限
SELECT p.permission_code, p.permission_name, p.category
FROM role_permissions rp
JOIN permissions p ON rp.permission_id = p.permission_id
JOIN AspNetRoles r ON rp.role_id = r.Id
WHERE r.Name = 'Doctor'
ORDER BY p.category, p.sort_order;

-- 查看权限变更日志
SELECT * FROM permission_change_logs 
ORDER BY operated_at DESC 
LIMIT 10;

-- 查看活跃的临时权限
SELECT * FROM temporary_permissions 
WHERE status = 'Active' 
AND valid_to > NOW()
ORDER BY granted_at DESC;
```

## 🐛 故障排查

### 问题1：迁移失败 - 文件被锁定
```
错误: The process cannot access the file because it is being used by another process
```
**解决**: 停止所有运行中的应用程序实例

### 问题2：外键约束错误
```
错误: Cannot add foreign key constraint
```
**解决**: 确保引用的表已存在，检查表名和字段名是否正确

### 问题3：权限数据初始化失败
```
错误: Duplicate entry for key 'IX_Permission_Code'
```
**解决**: 权限代码重复，检查 PermissionConstants 中是否有重复定义

### 问题4：角色权限分配失败
```
错误: Cannot insert NULL into column 'role_id'
```
**解决**: 确保角色已创建，检查角色名称是否正确
