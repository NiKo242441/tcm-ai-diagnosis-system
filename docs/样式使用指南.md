# 中医数字化诊疗平台 - 样式使用指南

## 概述

本指南说明如何在项目中使用基于PlainAdmin主题的统一医疗系统样式，确保整个平台的视觉一致性和用户体验。项目采用PlainAdmin的现代化设计风格，结合中医诊疗系统的专业需求。

## 设计理念

- **现代化设计**：采用PlainAdmin的简洁现代风格
- **专业医疗感**：保持医疗系统的专业性和可信度
- **用户友好**：优化用户体验，提高操作效率
- **响应式设计**：适配各种设备和屏幕尺寸

## 主题配色方案

基于PlainAdmin主题优化的医疗系统专业配色方案：

### 主要色彩
- **主色调**: `#4F6FBF` - 柔和的医疗蓝，适合大面积使用
- **浅主色调**: `#6B85D1` - 用于悬停状态和次要元素
- **深主色调**: `#3A5299` - 用于激活状态和强调元素
- **强调色**: `#365CF5` - 仅用于小点缀和重要提示，吸引注意力

### 背景和文本色彩
- **背景色**: `#F8FAFC` - 页面背景色
- **卡片背景**: `#FFFFFF` - 卡片和内容区域背景
- **主文本色**: `#475569` - 主要文本内容
- **次要文本色**: `#64748B` - 辅助文本信息
- **边框色**: `#E2E8F0` - 主要边框和分割线

### 功能色彩（医疗系统专用）
- **成功色**: `#059669` - 深绿色，表示健康/正常状态
- **警告色**: `#D97706` - 深橙色，表示需要注意的情况
- **危险色**: `#DC2626` - 深红色，表示异常/紧急情况
- **信息色**: `#0284C7` - 深蓝色，表示提示信息

## 快速开始

### 1. 引入样式文件

在页面头部引入样式文件：

```html
<!-- 引入FontAwesome图标库 -->
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0/css/all.min.css">

<!-- 引入医疗系统样式 -->
<link rel="stylesheet" href="~/docs/medical-system-styles.css">
```

或者在Razor页面中：

```html
@{
    ViewData["Title"] = "页面标题";
}

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0/css/all.min.css">
<link rel="stylesheet" href="~/docs/medical-system-styles.css">
```

### 2. 基本页面结构

```html
<div class="medical-container">
    <div class="medical-card">
        <div class="medical-card-header">
            <i class="fas fa-user-md"></i>
            页面标题
        </div>
        <div class="medical-card-body">
            <!-- 页面内容 -->
        </div>
    </div>
</div>
```

    <div class="col-md-6">

### 移动端适配

样式已包含响应式设计，在移动端会自动调整：
- 卡片内边距减小
- 按钮尺寸调整
- 紧凑型选择框在小屏幕上恢复全宽

## 最佳实践

### 1. 布局原则

- 使用 `medical-container` 作为页面容器
- 使用 `medical-card` 组织内容块
- 使用 `medical-section` 分组相关内容
- 合理使用栅格系统进行布局

### 2. 表单设计

- 为所有表单控件添加 `medical-form-control` 类
- 使用 `medical-form-label` 统一标签样式
- 对于内容较短的下拉框添加 `compact` 类
- 合理使用 `row g-3` 控制间距

### 3. 颜色使用

- 主要操作使用医疗蓝色系
- 状态指示使用语义化颜色
- 保持整体色调的一致性

### 4. 图标使用

- 每个功能区域使用相应的图标
- 保持图标风格统一
- 适当使用图标增强用户体验

## 自定义扩展

### 添加新的组件样式

如需添加新的组件样式，请遵循以下命名规范：

```css
.medical-[组件名] {
    /* 基础样式 */
}

.medical-[组件名]-[修饰符] {
    /* 修饰样式 */
}
```

### 颜色变量

可以通过修改CSS变量来调整整体配色：

```css
:root {
    --medical-primary: #2c5aa0;      /* 主色调 */
    --medical-secondary: #4a6fa5;    /* 辅助色 */
    --medical-accent: #1e4a8c;       /* 强调色 */
    /* 其他颜色变量... */
}
```
