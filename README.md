# 中医数字化诊疗平台 (TCM AI Diagnosis Platform)

## 📋 项目简介

中医数字化诊疗平台是一个基于 .NET 8 的智能化SaaS平台，旨在为中医医疗机构提供完整的数字化诊疗解决方案。平台采用多租户架构，集成了人工智能技术，能够辅助中医师进行证候分析、疾病诊断和治疗方案制定，同时提供完善的患者管理、病历管理和知识库管理功能。

### 核心特性

- 🏥 **多租户架构** - 支持多个医疗机构独立使用，数据完全隔离
- 🤖 **AI辅助诊断** - 基于机器学习的智能证候分析和疾病识别
- 📝 **智能医案生成** - 自动生成标准化的中医医案记录
- 📊 **就诊系列管理** - 完整的患者治疗过程跟踪和管理
- 📚 **中医知识库** - 完整的中医理论、方剂、药材知识体系
- 🔐 **完善的权限系统** - 基于RBAC的细粒度权限控制
- 📱 **响应式设计** - 支持PC、平板、手机等多种设备访问

### 技术栈

#### 后端技术
- **框架**: .NET 8

## 🚀 核心功能模块

### 1. 多租户管理
- 租户创建、配置和管理
- 数据完全隔离，确保安全性
- 灵活的服务套餐配置
- 资源配额管理

### 2. 用户与权限管理
- 支持4种核心角色：管理员、医生、患者、药剂师
- 基于RBAC的细粒度权限控制
- 110+权限点，覆盖所有业务场景
- 支持临时权限授予（会诊、转诊场景）
- 完整的审计日志记录

### 3. 患者管理
- 完整的患者档案管理
- 支持跨租户患者信息共享
- 患者搜索和高级筛选
- 患者与医生关联管理

### 4. 诊疗管理
- 就诊记录创建和管理
- 四诊信息结构化采集（望、闻、问、切）
- AI辅助证候分析和疾病识别
- 诊断结果确认和修改

### 5. 治疗方案管理
- 智能治疗方案生成
- 中药处方开具（支持配伍禁忌检查）
- 非药物疗法记录（针灸、艾灸、拔罐等）
- 个性化饮食和生活方式建议
- 方案版本化管理
- 风险警告和确认机制

### 6. AI医案管理
- 基于就诊系列自动生成AI医案
- 证候演变分析
- 治疗效果评估
- 诊断建议和预后评估
- 医生审核和确认机制

### 7. 就诊系列管理
- 围绕主诉组织连续诊疗过程
- 智能系列关联建议
- 系列进展分析和可视化
- 自动结束机制

### 8. 知识库管理
- 中医理论知识库
- 方剂数据库
- 中药数据库
- 穴位经络数据库
- 知识图谱构建

## 🔐 权限系统

### 权限架构
- **权限总数**: 110个
- **角色数量**: 4个（管理员、医生、患者、药剂师）
- **权限模式**: RequireAll（需要全部权限）/ RequireAny（需要任一权限）
- **临时权限**: 支持会诊、转诊等特殊场景
- **多租户支持**: 原生支持多租户隔离

### 权限分布

| 角色 | 权限数量 | 主要职责 |
|------|---------|---------|
| 管理员 (Manager) | 23 | 租户管理、用户管理、系统配置 |
| 医生 (Doctor) | 56 | 诊疗操作、治疗方案、病历管理 |
| 患者 (Patient) | 10 | 查看个人信息、诊疗记录、治疗方案 |
| 药剂师 (Pharmacist) | 6 | 查看处方、中药知识库 |

### 权限特性
- ✅ 智能缓存（30分钟过期）
- ✅ 完整审计日志
- ✅ 白名单路径支持
- ✅ 可配置启用/禁用
- ✅ 401/403标准错误响应

## 🛠️ 快速开始

### 环境要求
- .NET 8 SDK
- MySQL 8.0+
- Redis (可选)
- Docker (可选)

### 安装步骤

1. **克隆项目**
```bash
git clone https://github.com/your-repo/TcmAiDiagnosis.git
cd TcmAiDiagnosis
```

2. **配置数据库**
```bash
# 修改 appsettings.json 中的数据库连接字符串
"ConnectionStrings": {
  "Connection": "Server=localhost;Port=3306;Database=tcm-ai-diagnosis;User=root;Password=your_password;"
}
```

3. **运行数据库迁移**
```bash
cd TcmAiDiagnosis.Web
dotnet ef database update
```

4. **启动应用**
```bash
dotnet run
```

5. **访问应用**
```
http://localhost:5226
```

### Docker部署

```bash
# 构建镜像
docker build -t tcm-ai-diagnosis .

# 运行容器
docker run -d -p 5226:80 --name tcm-app tcm-ai-diagnosis
```
